### Prerequisites

- React
- Node.JS

### Usage

Run
yarn add use-data-fetch
OR
npm i use-data-fetch

### Usage in React

import useDataFetch from "use-data-fetch"
const data = useDataFetch(url) - `url` is the api's endpoint
